# cli-dojo 要件定義

## 機能要件

### Vim練習

- 問題文で指示された編集操作をVimキーバインドで実行する
- 初期テキストと目標テキストを提示し、一致したら正解とする
- キーストロークを記録し、操作履歴として表示する
- 問題はカテゴリ別に分類する（移動、編集、検索・置換、ビジュアルモードなど）
- 各問題に難易度（初級・中級・上級）を設定する

### Shell練習

- 問題文で指示されたタスクに対し、コマンドを入力して実行する
- 実行結果（stdout/stderr/終了コード）を期待値と比較し正誤判定する
- サンドボックス環境で実行する（ホスト環境を汚さない）
- 問題はカテゴリ別に分類する（ファイル操作、テキスト処理、パイプライン、プロセス管理など）

### 進捗管理

- 問題ごとの正答/不正答を記録する
- カテゴリ別の達成率を表示する
- 直近の学習履歴を一覧表示する

## 非機能要件

- ローカル完結（ネットワーク不要で動作する）
- macOS上のNode.js 20+で動作する
- Node/npmのバージョン管理はVoltaを使用し、package.jsonで固定する
- シェル実行はchild_process.execを使用する（node-ptyは使わない）
- データはlocalStorageに保存する（DB不要）
- 起動は `npm run dev` のみ
- レスポンス: 操作入力に対し100ms以内に画面反映

## スコープ

### やること

- Vimの基本〜中級操作の練習機能
- シェルコマンド（bash）の練習機能
- 問題セットの静的定義（JSONファイル）
- ローカルでの進捗保存

### やらないこと

- ユーザー認証・マルチユーザー対応
- 問題の動的生成（AI生成など）
- リモートデプロイ・ホスティング
- Vim上級機能（マクロ、Vimscript）
- zsh/fish固有の構文
- モバイル対応

## 画面構成・遷移

```
[トップ] ─┬─ [Vim練習]
           │     ├─ カテゴリ選択
           │     └─ 問題画面（エディタ + 問題文 + 判定結果）
           │
           ├─ [Shell練習]
           │     ├─ カテゴリ選択
           │     └─ 問題画面（ターミナル + 問題文 + 判定結果）
           │
           └─ [進捗確認]
                 └─ カテゴリ別達成率・履歴
```

### 各画面の概要

| 画面 | 内容 |
|------|------|
| トップ | モード選択（Vim / Shell / 進捗確認） |
| カテゴリ選択 | カテゴリ一覧と達成率の表示、問題開始 |
| Vim問題画面 | 問題文、初期テキスト付きVimエディタ、目標テキスト、判定ボタン |
| Shell問題画面 | 問題文、コマンド入力ターミナル、実行結果、期待出力との比較 |
| 進捗確認 | カテゴリ別の達成率グラフ、直近の学習履歴リスト |
