## 概要

Vim操作とShell操作の練習ができるローカルWebアプリケーション。
インフラ業務で頻出するVimキーバインドやシェルコマンドを、実践的な問題形式で反復練習できる。

## 目的

- Vimの基本〜中級操作を体で覚える
- シェルコマンド・スクリプトの書き方を練習する
- ローカル環境で気軽に反復できる

## 対象ユーザー

- インフラ業務でVim・シェルを日常的に使う開発者

## 主要機能

### 1. Vim練習モード

- 指定されたテキスト編集タスクをVimキーバインドで解く
- 例: 「カーソルを3行目に移動せよ」「単語を削除せよ」「行をヤンクして貼り付けよ」
- キーストロークの記録・正誤判定
- カテゴリ別の問題セット（移動、編集、検索・置換、レジスタなど）

### 2. Shell練習モード

- 与えられたお題に対して正しいコマンド・スクリプトを入力する
- 例: 「/var/log以下の.logファイルを新しい順に5件表示せよ」
- 入力したコマンドを実際に実行し、期待される出力と比較して正誤判定
- カテゴリ別の問題セット（ファイル操作、テキスト処理、プロセス管理など）

### 3. 進捗管理

- 問題ごとの正答率・所要時間の記録
- カテゴリ別の達成度表示

## 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | Next.js (App Router) |
| 言語 | TypeScript |
| Vimエミュレーション | CodeMirror 6 + @replit/codemirror-vim |
| ターミナルUI | xterm.js |
| シェル実行 | Node.js child_process / node-pty |
| スタイリング | Tailwind CSS |
| データ保存 | ローカルファイル (JSON) or localStorage |

## 動作環境

- ローカルのみ（デプロイ不要）
- macOS
- Node.js 20+
